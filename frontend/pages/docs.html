Getting Started

AmkyawDev AI Agent is an intelligent assistant powered by advanced AI models.

npm install @amkyawdev/ai-agent

Quick Start

const agent = new AIAgent({
    apiKey: 'your-api-key',
    model: 'mixtral-8x7b-32768'
});

const response = await agent.chat('Hello!');

API Reference

Method Description
chat(message) Send a message and get AI response
search(query) Search the web for information
analyze(file) Analyze uploaded files
