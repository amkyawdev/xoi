Getting Started

Quick introduction to AmkyawDev AI Agent

AmkyawDev AI Agent is an intelligent assistant powered by advanced AI models. It provides natural language understanding, code assistance, web search, and more.

pip install amkyawdev-agent

Quick Start

Get up and running in minutes

from amkyawdev import AIAgent

agent = AIAgent(api_key="your-api-key")
response = await agent.chat("Hello!")
print(response)

API Reference

Complete API documentation

Method Description
chat(message) Send a message and get AI response
search(query) Search the web for information
analyze(file) Analyze uploaded files
stream(message) Streaming chat responses
